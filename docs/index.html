Dataset Overview

A benchmark for evaluating AI biomedical peer review tools

bioreview-bench pairs published open-access biomedical articles with the substantive concerns that professional peer reviewers raised during journal review. Each concern is annotated with category, severity, and an outcome-anchored author stance label. The benchmark measures how well AI tools identify the concerns that human reviewers would raise.
